# SenseGNAT Logo Kit Brief

## Deliverables intended

- mascot-only mark
- wordmark
- combined lockup
- monochrome variants
- small-icon shield mark
- favicon/app icon treatment

## Combined lockup direction

A practical first lockup can be:

- mascot above or left of the wordmark
- `SenseGNAT` in bold dark ink
- `Behavior is the signal.` as optional supporting line
- shield signal emblem reused as a product icon

## Usage notes

Use the mascot for marketing, docs, presentations, and the GNAT-o-sphere visual system. Use the shield-only icon for product navigation, repository badges, and application chrome.

## Prompt seed for future illustration refinement

Create a fun mascot in the style of the existing GNAT character. Preserve the same face, body silhouette, wings, and friendly expression. Replace the red suit with a deep purple suit and dark ink accents. Put the mascot in a focused predictive pose with both fingers touching the sides of its head, as if analyzing network signals. Keep the chest shield but replace the lock-and-code emblem with a clean signal-wave emblem. Add subtle purple signal arcs around the head. The result should feel analytical, observant, and premium, not magical or aggressive.
